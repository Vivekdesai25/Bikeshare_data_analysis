# US Bikeshare Data Analysis

## Overview
This project performs exploratory data analysis on bikeshare datasets from major US cities using Python and pandas. It provides insights into travel behavior through an interactive command-line application.

## Key Features
- Filter data by city, month, and day
- Analyze travel trends (most common month, day, and hour)
- Identify popular start and end stations
- Determine most frequent trip combinations
- Calculate total and average trip durations
- Explore user information such as user types, gender, and birth year (where available)
- Interactive command-line interface for user-driven analysis
- Display raw data in batches of five rows
- Robust input validation and error handling

## Dataset
Sample datasets (first 200 rows) are included for demonstration purposes due to size limitations. The data contains trip details such as start time, end time, stations, trip duration, and user information.
